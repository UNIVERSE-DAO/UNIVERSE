# COMMON

> _Why we gather._  
> _Why we care._  
> _Why it matters._
<br>

COMMON is the shared field of meaning,  
where values, culture, and collective resonance become visible.

Not enforced.  
Not owned.  
But unmistakably felt.

the shared WHY beneath participation.

## WHAT

COMMON reflects the symbolic foundations that allow systems to align:
- culture,
- belief,
- ritual,
- contribution,
- and shared purpose.

It transforms meaning into belonging,  
while preserving coherence throughout the Infinite Game.

<br>

| Element | Description |
|---|---|
| **Belief** | Shared values that shape participation |
| **Culture** | Rituals, symbols, language, and emotional resonance |
| **Cause** | Collective direction rooted in meaning |
| **Contribution** | Participation aligned with shared intention |
| **Ritual** | Repeated acts that reinforce coherence |
| **Resonance** | The social signal of aligned understanding |

## HOW

COMMON emerges through shared experience, expression, and participation.

As people gather around meaningful signals,  
culture becomes a living coordination layer.

The system does not manufacture belonging.  
It helps reveal shared resonance.
